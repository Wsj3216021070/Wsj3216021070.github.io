首先任意浏览器搜索mysql，如图

选择download

下拉找到这个


这里出现两个下载渠道，其中①为压缩包，②为直接下载客户端（我试了老半天①终于成功之后发现②挺快的…）

首先是①的安装方法

1、选择下载后会在你选择的安装路径下出现这么一个压缩包

2、解压到当前文件夹，打开后得到如下

其中data和my.ini是没有的，需要我们自己手动添加

3、新建文本文档，直接改头换面，后缀也改掉，改成my.ini，这时候提示警告，我们不用搭理它，确定就可以了。接下来打开这个文本文档，将以下内容复制进去

[mysqld]

设置3306端口

port=3306

设置mysql的安装目录

basedir=E:\MySQL\mysql-8.0.28-winx64

设置mysql数据库的数据的存放目录

datadir=E:\MySQL\mysql-8.0.28-winx64\data

允许最大连接数

max_connections=200

允许连接失败的次数。这是为了防止有人从该主机试图攻击数据库系统

max_connect_errors=10

服务端使用的字符集默认为UTF8

character-set-server=utf8

创建新表时将使用的默认存储引擎

default-storage-engine=INNODB

默认使用“mysql_native_password”插件认证

default_authentication_plugin=mysql_native_password
[mysql]

设置mysql客户端默认字符集

default-character-set=utf8
[client]

设置mysql客户端连接服务端时默认使用的端口

port=3306
default-character-set=utf8

其中一定要记得修改basedir、datadir的路径（看你自己安装在哪里啦），至此，my.ini部分完成

4、设置环境变量
右击 此电脑–>属性–>高级系统设置–>环境变量

这里一步一步来

确定后双击Path，新建这两个哈

每一步都一定一定记得点确定！！！！！（别问我为什么那么强调…）

5、创建data文件夹（不要自己在刚刚那里创建哈，建了的删掉就可以，不然mysql的服务可能启动不了）
搜索cmd，打开面板，切记一定要用管理员身份打开，不然很多地方走不通

输入 mysqld –initialize-insecure 回车，等个两秒钟命令就执行完了，再打开文件夹就会出现data文件夹

6、初始化语句
把 mysqld –defaults-file=E:\Mysql\mysql-8.0.28-winx64\my.ini –initialize –console 复制进去，回车
这里风平浪静，什么也没发生

7、安装MySQL
把 mysqld install 复制进去，回车
会出现一行简短的 Service successfully installed 就成啦

8、初始化MySQL
mysqld –initialize-insecure –user=mysql
显示 MySQL 服务正在启动…
MySQL 服务已经启动成功.
右击 此电脑–>属性，这样子点下去，下滑就可以找到MySQL的服务，这里默认自启动就好，不然每次打开很麻烦

9、启动MySQL
mysqladmin -u root -p password 123456
这里的“123456”为密码，可以改为任意的你想要的密码，而用户名都是“root”，无法更改
回车后出现Enter password，咱不用鸟它，回车就好了

10、登录，进入mysql界面
mysql -u root -p123456 回车即可登录，不过这时密码是可视化的，如果我们旁边有人咧？这样子👇
