五、单行处理函数（一个输入对应一个输出）

1、常见单行处理函数：

lower 转换小写
  如：mysql> select lower(ename) as ename from emp;

upper 转换大写
  如：mysql> select upper(ename) as ename from emp;

substr 取子串，格式：substrate（被截取的字符串，起始下标，截取的长度）
下标都是从1开始的
  如：mysql> select substr(ename, 1, 1) as ename from emp;

length 取长度，格式：length(被取的字符串)
  如：mysql>  select length(ename) enamelength from emp; （空格表省略的 as ，为起别名）

concat 字符串拼接，格式：concat（拼接字符串1，拼接字符串2）
拼接后为12，中间无空格
如mysql>  select concat(empno,ename) from emp;

trim 去空格，格式：trim（）
其()内的所有空格都将去除
如：mysql> select * from emp where ename = '  KING';
       Empty set (0.00 sec)
       mysql> select * from emp where ename = trim('  KING');

       +---------+---------+--------------+------+--------------+---------+--------+----------+
       1 row in set (0.00 sec)

round 四舍五入，格式：round（数字1，数字2）
数字1为要四舍五入的数，数字2为小数点后保留几位数
如：round（1234.56，1）结果为1234.6
       round（1234.56，0）结果为1235
       round（1234.56，-1）结果为1240

rand（） 生成随机数，格式：rand（）
随机生成0~1的数字
如：随机生成0~100的整数
       mysql> select round(rand()*100,0) from emp;

ifnull 转化null，格式：ifnull（A，B）
其中A为参与的那一列数据，将null转化为B（有null参与的任何计算结果都为null，如1+null=null，与1+0=1不同）
如：mysql> select ename,(sal*12+comm) as yearsal from emp;

case 如果，格式：case ... when ... then ... else end
其中when ... then ... 的个数按需求给，不变的部分用else，类似于 C 的 if 语句
如：mysql> select ename,job,sal as oldsal,(case job when 'MANAGER' then sal*1.1 when 'SALESMAN' then sal*1.5 else sal end) as newsal from emp;

找出员工名字首字母为 A 的员工信息：
①：模糊查询
mysql> select ename from emp where ename like ‘A%’;
②：substr函数
mysql> select ename from emp where substr(ename,1,1) = ‘A’;

将名字首字母改为大写（原本全为小写）
运用了函数的嵌套
①select upper(substr(name,1,1)) from t_student; 取学生名字首字母并改为大写
②select substr（name,2,length(name) - 1） from t_student; 取学生学生去掉首字母的名字保持不变
③select concat（upper(substr(name,1,1))，substr（name,2,length(name) - 1)）as result from t_student;拼接两部分并起名为result，的学生名字首字母大写后的结果
